A completely custom-made website for my longest-running client, Paradise in a Box. The website, utilizing a whole set of technologies, allows users to create and customize their own products. Individual customers can pay for their custom products and have them shipped, using the Stripe API. Large customers can place bulk orders, which are processed differently.
