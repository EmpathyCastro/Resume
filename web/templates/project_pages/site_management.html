Project Site Management Utility

{% include "project_pages/carousel.html" %}

Created a web app for my Computer Science Capstone project as part of a team of six. The app enables users to create and manage project sites, which are used to track zones, materials, personnel, etc. within a construction project. Contributed to the frond-end development of the app, using React, Axios, and Bootstrap extensively.

Features
  • Boundary creation through interactions with map using Leaflet (a JavaScript map library).
  • Create, edit, and move resources within projects.
  • Show on-screen notifications when resources are modified.
  • Deployed using AWS.
