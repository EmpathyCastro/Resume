CUChange: Ideal ETL

{% include "project_pages/carousel.html" %}

Developed multiple web apps for CUChange, a medical research center affiliated with the University. This was the biggest and most complex. The team had inherited a legacy Extract, Transform, Load (ETL) system that was complex and difficult to maintain. This project created a user-friendly ETL system, using a universal process to extract data from any study, transform it into a standard format, and load it into the laboratory's database.

Features
  • User authentication using up-to-date security practices, such as salt+hash passwords.
  • Includes a universal Extract, Transform, Load process.
  • Multi-field record search.
