Phi Sigma Pi: Attendance Management

{% include "project_pages/carousel.html" %}

When I was elected the Chapter Secretary of Phi Sigma Pi, I took it upon myself to overhaul our attendance tracking process. Previously, we had been using a Google Forms and Sheets system, which took a fair amount of work to maintain and mistakes were easy to make. I created a web app that allowed members to submit attendance for meetings and events, and the officers to view and manage the fraternity's attendance records. The app allows officers to view attendance records for individual members, as well as for the entire fraternity. Additionally, members are able to see their progress towards the fraternity's attendance requirements.

Features
  • User authentication using up-to-date security practices, such as salt+hash passwords.
  • Permission system to give individual access to users for certain pages.
  • Email notifications using the Gmail API.
Technologies
Heroku
Python + Flask
Bootstrap
MongoDB
Gmail API
