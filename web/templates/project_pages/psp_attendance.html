Phi Sigma Pi: Attendance Management

{% include "project_pages/carousel.html" %}

Elected Chapter Secretary of Phi Sigma Pi. Initiated the overhaul of the attendance tracking process, transforming it from a cumbersome Google Forms and Sheets system to an efficient and user-friendly web app. The app enables members to submit attendance for meetings and events and monitor progress toward attendance requirements. Fraternity officers can view and manage individual and comprehensive records.

Features
  • User authentication using up-to-date security practices, such as salt+hash passwords.
  • Permission system to give individual access to users for certain pages.
  • Email notifications using the Gmail API.
Technologies
Heroku
Python + Flask
Bootstrap
MongoDB
Gmail API
